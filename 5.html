Завдання 5:

Знайти частку K=L/M, де L – кількість нульових елементів масиву B(8), а M – кількість нульових елементів масиву C(8).

Лістинг програми:
$B=array(0,9,4,0,3,4,0,7);
$L=0;
$C=array(7,6,3,0,89,7,5,0);
$M=0;
foreach ($B as &$item){
    if($item==0){
        $L++;
    }
}
foreach ($C as &$item) {
    if ($item == 0) {
        $M++;
    }
}
$K=$L/$M;
echo "\nFifth task: K=".$K;
